# Redes de Computadores

Este repositório reúne projetos de **Redes de Computadores** desenvolvidos no **Cisco Packet Tracer**, com o objetivo de praticar, documentar e demonstrar conhecimentos em configuração, simulação e análise de redes.

Os projetos abordam conceitos fundamentais e intermediários de redes, incluindo endereçamento **IPv4** e **IPv6**, roteamento, **DHCP**, **NAT**, **DNS**, **VLANs**, comunicação entre redes, testes de conectividade e configuração de dispositivos como roteadores, switches, servidores e hosts.

## Objetivo

O objetivo deste repositório é armazenar e apresentar projetos práticos de redes, servindo como portfólio técnico e material de estudo para evolução na área de infraestrutura, redes de computadores e engenharia de computação.

## Tecnologias e Ferramentas Utilizadas

* Cisco Packet Tracer
* IPv4
* IPv6
* DHCP
* NAT
* DNS
* Roteamento estático e dinâmico
* VLANs
* Switches e roteadores Cisco
* Servidores e hosts simulados

## Conteúdos Abordados

Neste repositório, os projetos podem incluir:

* Criação de topologias de rede
* Configuração de endereçamento IPv4 e IPv6
* Sub-redes e planejamento de endereçamento
* Configuração de DHCP para distribuição automática de IP
* Configuração de NAT para tradução de endereços
* Configuração de DNS e servidores
* Roteamento entre redes
* Comunicação entre LANs e WANs
* Testes com ping, traceroute e comandos de verificação
* Análise e correção de problemas de conectividade
